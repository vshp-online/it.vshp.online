# УП.11 - Учебная практика модуля "ПМ.11 - Разработка, администрирование и защита баз данных"

## Тематический план

- [01 - Анализ структуры базы данных `world`.](./practicals/01_practical.md)
- 02 - Выполнение сложных запросов SELECT к базе `world`: использование JOIN для объединения таблиц, агрегатных функций (SUM, AVG, COUNT), группировки по регионам и континентам, подзапросы и оконные функции для анализа населения и языков.
- 03 - Модификация данных в базе `world`: операции INSERT (добавление новых городов), UPDATE (обновление информации о странах), DELETE (удаление устаревших записей); управление транзакциями (COMMIT, ROLLBACK, SAVEPOINT).
- 04 - Анализ структуры базы данных `employee`: изучение таблиц employees, departments, dept_emp, dept_manager, salaries, titles, связей и ключей.
- 05 - Выполнение сложных запросов SELECT к базе `employee`: JOIN для связи сотрудников с отделами и зарплатами, агрегатные функции для расчёта средней зарплаты по отделам, подзапросы для поиска сотрудников с максимальной зарплатой, оконные функции для ранжирования.
- 06 - Модификация данных в базе `employee`: операции INSERT (приём новых сотрудников), UPDATE (изменение должностей и зарплат), DELETE (увольнение); управление транзакциями и целостностью данных.
