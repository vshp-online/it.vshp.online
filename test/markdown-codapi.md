## Python в браузере

### С отображением line-numbers (по-умолчанию)

::: preview Посмотреть код

  ::: play sandbox=python editor=basic

  ```python
  # Комментарий
  print("Hello from Python!")
  ```

  :::

:::

### Без отображения line-numbers

::: preview Посмотреть код

  ::: play sandbox=python editor=basic

  ```python :no-line-numbers
  # Комментарий
  print("Hello from Python!")
  ```

  :::

:::
